Description/Source

This date set called Athlete events from the free data set website www.keggle.com is a collection of data of ever Olympics medal winner from 1896-2016. The data is organized into 15 columns that contain information from simple info about the athlete’s sex to his height, weight and age. The date has a lot of N/A values, I tried to use R-stuido to clean up the date but I wanted to check it out in more detail, so I used Excel and opened it as a CSV file and played with it. All in all, it is a very clean and nice date set. In my archived folder submitted via blackboard you will find the original file and the cleaned one that I used. There weren’t to many lines of N/A values when compared to the file over all, and they were in places that didn’t matter to the questions I was asking. So instead of Just having it I decide to remove all the lines with N/A and was left with a much cleaner dataset.
